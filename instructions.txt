MC Server Share is now a desktop app plus coordinator API.

1. Install Node.js 24 LTS, Rust stable, Java, and the playit.gg client.
2. Run `npm install`.
3. Configure `apps/coordinator/.env.local` from `apps/coordinator/.env.example`.
4. Apply `apps/coordinator/db/schema.sql` to Neon Postgres.
5. Run the coordinator with `npm run dev --workspace @mc-share/coordinator`.
6. Run the desktop app with `npm run dev --workspace @mc-share/desktop`.
7. In the desktop app, import a Minecraft Java server folder, create or load a
   share, claim the host lock, click Host, then Stop and Upload when finished.

For Vercel:

1. Import `nnicholas-c/mc-server-share-app`.
2. Set Root Directory to `apps/coordinator`.
3. Add Neon, Vercel Blob, and the env vars from `apps/coordinator/.env.example`.
4. Send friends `https://your-coordinator.vercel.app/share/<code>`.

Friends install the desktop app once, click "Open In MC Server Share" from the
share page, click Download Latest, then Host. The `mcservershare://` deep link
loads the coordinator URL and share code automatically.

The old `server1.20.4` folder is local-only and ignored by Git. Import it from
the desktop app if you want to publish the current RLCraft/Dregora server.
